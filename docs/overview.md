# Overview

VFA Hook is a volatility derivatives protocol built around three execution domains:

- Origin chain Uniswap v4 pools and hook telemetry
- Reactive Network automation for epoch completion
- Destination chain settlement executor + market accounting

Users trade long-vol or short-vol positions per epoch.
